# LevelXp

## Опис
LevelXp — це Django-проєкт для соціальної платформи з чатами, профілями та постами.

## Встановлення та залежності

### Вимоги
- Python >= 3.8
- PyTorch >= 1.7.1
- numpy

### Встановлення
```bash
pip install -r requirements.txt
```
